## 1. 产品概述

高端连锁酒店PMS（物业管理系统）与客控系统，实现从预订到离店的全流程智能化管理。系统通过会员偏好智能排房、IoT客控自动化、人脸识别自助入住、重力感应迷你吧、智能排房与服务调度等能力，打造无缝的酒店运营体验。

- 目标用户：高端连锁酒店运营团队（服务员、前台、客房主管、总经理）
- 核心价值：提升运营效率、降低人工成本、增强会员体验、数据驱动决策

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 服务员 | 工号+密码 | 接收服务请求推送、更新房态、查看个人任务 |
| 前台 | 工号+密码 | 办理入住/退房、查看房态、处理预订、管理账单 |
| 客房主管 | 工号+密码 | 管理保洁任务、房态审核、排班管理、服务质量监控 |
| 总经理 | 工号+密码 | 查看经营大屏、夜审报表导出、全量数据查看、系统配置 |

### 2.2 功能模块

1. **登录页**：角色选择、身份验证、权限分流
2. **工作台首页**：角色定制化概览、快捷操作入口、待办事项提醒
3. **房态管理**：房间网格视图、实时状态（空房/住客/待清洁/维修）、IoT设备控制面板
4. **预订与入住**：新建预订、智能排房、自助入住模拟、人脸识别验证、发卡/蓝牙钥匙
5. **客务服务**：服务请求列表、智能派工（最近服务员）、迷你吧消费、任务状态追踪
6. **客房管理**：保洁任务看板、扫码净房、质量检查、排班管理
7. **会员中心**：会员列表、偏好管理、消费记录、自动升级、权益展示
8. **总经理大屏**：实时入住率、RevPAR、网评得分、能耗监控、趋势图表
9. **报表中心**：夜审报表、营收分析、一键导出

## 3. 核心流程

### 3.1 预订到入住全流程

客人在App提交预订 → 系统匹配会员偏好自动排房（高楼层/无烟/硬枕） → 入住前1小时系统下发IoT指令开启空调和灯光 → 客人到达自助入住机 → 人脸识别核验身份 → 系统自动发卡或授权手机蓝牙钥匙 → 入住完成

```mermaid
flowchart TD
    "客人App预订" --> "系统匹配会员偏好"
    "系统匹配会员偏好" --> "智能排房推荐"
    "智能排房推荐" --> "入住前1小时"
    "入住前1小时" --> "自动开启空调灯光"
    "自动开启空调灯光" --> "客人到达酒店"
    "客人到达酒店" --> "自助入住机人脸识别"
    "自助入住机人脸识别" --> "身份核验通过"
    "身份核验通过" --> "自动发卡/蓝牙钥匙"
    "自动发卡/蓝牙钥匙" --> "入住完成"
```

### 3.2 住中服务流程

客人发起服务请求 → 系统自动匹配最近服务员 → 推送到服务员智能手表 → 服务员接受并执行 → 完成后更新状态；迷你吧重力感应自动记录 → 消费实时同步账单

```mermaid
flowchart TD
    "客人发起服务请求" --> "系统智能派工"
    "系统智能派工" --> "推送至最近服务员手表"
    "推送至最近服务员手表" --> "服务员接受任务"
    "服务员接受任务" --> "执行服务"
    "执行服务" --> "更新任务状态"
    "迷你吧取用物品" --> "重力感应检测"
    "重力感应检测" --> "自动记录消费"
    "自动记录消费" --> "同步至账单"
```

### 3.3 退房流程

客人发起退房 → 系统自动查房（检查迷你吧消费、物品完好） → 无异常秒级退房 → 自动开具电子发票 → 会员积分累计 → 消费金额累计检查升级

```mermaid
flowchart TD
    "客人发起退房" --> "系统自动查房"
    "系统自动查房" --> "检查迷你吧消费"
    "检查迷你吧消费" --> "检查物品完好"
    "检查物品完好" --> "{是否有异常}"
    "{是否有异常}" -->|"无异常"| "秒级退房"
    "{是否有异常}" -->|"有异常"| "通知前台处理"
    "秒级退房" --> "开具电子发票"
    "开具电子发票" --> "会员积分累计"
    "会员积分累计" --> "检查升级资格"
```

### 3.4 保洁流程

客人退房 → 房态变为"待清洁" → 系统分配保洁员 → 保洁员打扫完毕扫码 → 房态更新为"净房" → 通知前台可售

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝灰（#1B2A4A）+ 金色点缀（#C9A96E），传达高端奢华感
- **辅助色**：暖灰（#F5F0EB）、浅金（#E8D5B0）、状态绿（#34D399）、警示红（#EF4444）
- **按钮风格**：圆角8px，金色主按钮带微妙渐变，次要按钮为描边风格
- **字体**：标题使用 Playfair Display（奢华衬线体），正文使用 DM Sans（现代无衬线体）
- **布局风格**：左侧固定导航栏 + 顶部信息栏，内容区域卡片式布局
- **图标风格**：Lucide Icons 线性图标，与整体简约奢华风格统一
- **背景纹理**：微妙的噪点纹理覆盖，深色模式为主

### 4.3 响应式设计

- 桌面端优先（1920×1080为主），适配1440px和1366px
- 总经理大屏优化4K显示
- 平板端（1024px）侧边栏可折叠
- 移动端（768px以下）切换为底部导航 + 卡片堆叠布局

### 4.4 动效设计

- 页面切换：淡入淡出 + 轻微上滑
- 卡片交互：hover上浮 + 阴影加深
- 数据更新：数字滚动计数动画
- 房态变化：颜色渐变过渡
- IoT控制：开关状态弹性动画
- 大屏刷新：图表数据平滑过渡动画
